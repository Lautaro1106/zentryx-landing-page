# Zentryx - Landing Page (Astro)

Landing page profesional para Zentryx, enfocada en servicios de automatización de procesos. Construida con Astro y Tailwind CSS.

## Características

- **Stack Moderno**: Astro 5 + Tailwind CSS 4.
- **Diseño Responsive**: Adaptado a todos los dispositivos.
- **Performance**: HTML estático optimizado.
- **Modo Oscuro**: Diseño dark theme con paleta Azul/Dorado.

## Estructura del Proyecto

```text
src/
├── components/       # Componentes UI (Hero, Proyectos, etc)
├── layouts/          # Layout base
└── pages/            # Página principal
```

## Instalación y Uso

1.  **Clonar el repositorio**:
    ```bash
    git clone https://github.com/TU_USUARIO/zentryx-astro.git
    cd zentryx-astro
    ```

2.  **Instalar dependencias**:
    ```bash
    npm install
    ```

3.  **Correr en desarrollo**:
    ```bash
    npm run dev
    ```

## Deploy en Cloudflare Pages / Vercel / Netlify
