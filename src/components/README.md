# Componentes UI

Esta carpeta contiene los bloques de construcción reutilizables de la interfaz.

## Componentes Principales

- **Header.astro**: Barra de navegación superior. Incluye el logo y botón de contacto.
- **Hero.astro**: Sección principal con el tagline animado y llamada a la acción.
- **Enfoque.astro**: Explica la metodología de trabajo (Proceso, Tecnología, Transparencia).
- **Proyectos.astro**: Grid de tarjetas que muestra los trabajos realizados.
- **LeadMagnet.astro**: Sección para descargar la "Guía de 7 Preguntas".
- **SobreMi.astro**: Sección personal con foto y biografía.
- **Footer.astro**: Pie de página con redes sociales y copyright.
- **WhatsAppButton.astro**: Botón flotante de contacto fijo en la esquina inferior.

## Notas
Todos los componentes son responsivos y utilizan clases de Tailwind CSS.
